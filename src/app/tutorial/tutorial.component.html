A Quick Tutorial

The Open menu item

The Open menu item takes you to the dialog where you can open your translation file by dragging and dropping it onto the marked screen area.

If you dropped a valid XLIFF translation file on the marked area, the application takes you to the translation dialog. This is where the main part of the translation work, basically all the work, is done.

The Synchronise menu item

If you happen to have a newly created or updated file with all the current content and an old, not up to date translation file, this dialogue might be helpful.

Drop the files on the corresponding areas and the application will merge the contents. New additions and existing translations where the source text has changed in the reference document will be marked for review, others will be accepted as okay. You will be redirected to the translation dialogue where you can review the translations and download the result.
